## 0. 설계 원칙 (브리프·레퍼런스 기반)

- **레이아웃 리듬**: 다크(#1a1a1a) ↔ 화이트(#f5f5f5) ↔ 블랙 섹션 교차 (MONENS 방식). 페이지마다 첫 섹션은 다크 히어로로 통일.
- **여백 중심**(VISUAL_DENSITY 3): 섹션당 정보 최소화, 큰 여백·큰 타이포.
- **시네마틱 모션**(MOTION 9): 히어로 텍스트 reveal(회색→흰색), 스크롤 진입 페이드/슬라이드, 패럴랙스 배경.
- **히어로/풀블리드 3레이어 구조**(v3·사령관 지시): 모든 풀영상/히어로 구간은 `[L1 배경영상(섹션 전체 채움)] → [L2 어두운 오버레이 딤(반투명 검정/다크 그라데이션)] → [L3 텍스트 오버레이(회색→흰색 reveal)]`. 영상 위에 어두운 딤을 깔아 텍스트 가독성·무게감 확보. 적용: HOME H1·H5 + 전 페이지 히어로 A1·B1·I1·S1·C1.
- **금융/투자 규제 준수**: 수익보장·과장 표현 금지. "정보 제공·교육"이라는 사실 기반 문구만. 카피는 다음 단계(페이/시엘)에서 확정 — 본 기획은 구조·슬롯에 집중.
- **이미지/영상**: 실제 자산 미정 → 전부 **슬롯 박스 + 라벨(크기·비율·용도)** 처리. 사령관이 직접 삽입.

## 2. 글로벌 요소 (전 페이지 공통)

### 2-1. 헤더 / 네비
- 고정(sticky) 헤더. 초기 투명 → 스크롤 시 다크 배경 fade-in.
- 구성: `[로고 슬롯 — 좌] · [HOME / 회사소개 / 사업소개 / 투자정보 / 고객센터 / 문의하기 — 중앙·우] · [문의하기 CTA 버튼 — 우]`
- 모바일: 햄버거 → 풀스크린 다크 오버레이 메뉴.
- 로고: **미정** → `[로고 슬롯 — 가로형, 약 140x32]`

### 2-2. 푸터 (다크 #1a1a1a)
- 구성: 회사명/로고 슬롯 · 사업영역 5 링크 · 빠른 메뉴(6) · 연락처(미정 슬롯) · 사업자 정보(미정) · 카피라이트
- **투자 면책 고지 영역**(고정): "본 사이트의 정보는 투자 참고용이며 투자 권유·수익 보장이 아닙니다. 투자 판단의 책임은 본인에게 있습니다." (규제 대응 — 카피 단계 확정)

### 2-3. Floating 요소
- 우하단 floating "문의하기" 버튼 (전 페이지).
- 우하단 "맨 위로↑" 버튼 (스크롤 시 등장).

## 3. 페이지별 섹션 구성표

### ▣ HOME  ★v2 임팩트 보강 (9단 시네마틱 흐름)★

> **보강 의도**: 기존 6단(소개→사업→강점→정보→CTA)이 교과서적·평탄 → 9단으로 **강(임팩트)·약(여백)·강** 리듬 부여. 풀블리드 스테이트먼트 2개를 호흡점으로 삽입하고, 사업영역은 정적 4카드 → 인터랙티브 가로 스크롤/hover 확장으로 다이나믹화. 숫자 스탯 카운터로 신뢰의 무게(WITHPLUS 골드+다크 무게감) 부여.
>
> **Before(v1, 6단)**: H1 히어로 → H2 한줄소개 → H3 사업 4카드 → H4 강점 → H5 정보 → H6 CTA
> **After(v2, 9단)**: H1 히어로 → **H2 스탯 카운터** → H3 한줄소개 → **H4 사업영역(인터랙티브)** → **H5 풀블리드 스테이트먼트** → H6 강점 → H7 정보 → **H8 풀블리드 클로징 스테이트먼트** → H9 CTA
